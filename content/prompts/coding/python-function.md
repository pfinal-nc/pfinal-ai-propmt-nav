作为一个资深Python专家，请帮我生成一个[function_name]函数，功能是[description]。

要求：

1. 类型注解：
   - 为所有参数和返回值添加完整的类型注解
   - 使用typing模块处理复杂类型（如Union, Optional, List等）

2. 文档字符串：
   - 遵循Google Python风格指南或NumPy风格
   - 包含函数描述、参数说明、返回值说明和异常说明
   - 提供使用示例

3. 错误处理：
   - 预见可能的异常情况并处理
   - 添加适当的参数验证
   - 抛出有意义的异常信息

4. 代码质量：
   - 遵循PEP8编码规范
   - 确保代码可读性和可维护性
   - 添加必要的注释（非文档字符串）
   - 考虑性能优化

5. 测试考虑：
   - 提供单元测试用例建议
   - 考虑边界条件和特殊情况

请确保生成的函数既实用又专业。
